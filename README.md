## Como executar

1. Baixe todos os arquivos em uma pasta ou clone o repositório.

2. Acesse a pasta do projeto

   ```bash
   cd BulaFacil (Ou diretório que está o aplicativo)
   ```
   
4. Instale as dependências

   ```bash
   npm install
   ```

5. Inicie o aplicativo

   ```bash
   npx expo start
   ```
